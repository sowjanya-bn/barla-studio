# Barla Studio starter

A lightweight Astro starter for a shared host with independent paths:

- `/sowjanya/` for a minimal programmer profile
- `/bhargav/` for an artist profile
- `/bhargav/photography/`
- `/bhargav/paintings/`
- `/bhargav/books/`

## Run locally

```bash
npm install
npm run dev
```

## Notes

- Replace placeholder emails and social links
- Swap placeholder SVGs in `public/images`
- Update `site` in `astro.config.mjs` before deploying to GitHub Pages
- You can keep `/` as a very light root page or later redirect it elsewhere

## GitHub Pages

A starter workflow is included in `.github/workflows/deploy.yml`
